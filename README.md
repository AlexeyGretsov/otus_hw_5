# otus_hw_5
OTUS C++ Professional home work 5. Editor

Command line options:
 - 'p' – draw point
 - 'l' – draw line
 - 'r' – draw rect
 - 'e' – draw ellipse
 - 't'[<space><any text>] – draw text
 - 'i' – draw image
 - 'rem' – remove last shape
 - 'n' – create new document
 - 'save'<space>'filename' - save to file
 - 'restore'<space>'filename' - restore from file
 - 'q' for quit
